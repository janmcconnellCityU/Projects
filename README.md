A collection of independent projects, data explorations, and technical reference materials.

This repository contains selected work created outside of my formal coursework, along with reusable reference notebooks developed from concepts, languages, and tools I use in data analysis and data science.

## 📊 Projects

### 📡 Radar Sample Project

Synthetic radar signal analysis exploring distance-based signal attenuation and anomaly detection. The project compares observed signal strength with expected behavior to identify unusually weak signals and evaluates the detection method against known injected anomalies.

### 🛰️ Satellite Orbit Data Exploration

Exploratory analysis of publicly available satellite orbital element data. The project examines orbital regimes, inclination patterns, orbital periods, and unusual or highly elliptical orbits to identify patterns in how modern satellites are distributed across common orbital corridors.

## 📚 Reference Notebooks

### 🐍 Python Cheat Sheet

A practical Python reference covering common syntax, data structures, NumPy, pandas, Matplotlib, scikit-learn, TensorFlow/Keras, and Jupyter Notebook commands.

### 🎨 CSS Colors

A Jupyter Notebook reference for CSS color names and values.

### 📈 R Reference Guide

A practical Jupyter notebook covering core R programming, data structures, data manipulation with tidyverse, visualization with ggplot2, statistical analysis, data cleaning, file handling, and common workflows using both built-in datasets and AdventureWorks sample data.
